JayEarn.com
🌟 About the Project
JayEarn.com is an online rewards platform where users can earn money by completing simple tasks such as:
Taking surveys
Playing games
Completing online offers
Referring friends
Users can withdraw their earnings via M-Pesa or PayPal.
🚀 Features
User registration and login system
Earnings dashboard
Survey and game sections
Referral program
Withdrawal request system
Black & gold modern UI design
💰 How It Works
Sign up for a free account
Complete tasks (surveys, games, offers)
Earn points for each activity
Refer friends to earn bonuses
Withdraw earnings via M-Pesa or PayPal
